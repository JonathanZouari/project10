## 📊 Oded's Streamlit Dashboard

**Oded's Dashboard** הוא דאשבורד אינטראקטיבי פשוט שנבנה עם `Streamlit` לצורך הדגמת הצגה וניתוח בסיסי של נתוני עובדים.

הדאשבורד כולל:
- **טבלת נתונים** עם שמות, גיל, מחלקה וציון.
- **גרף עמודות** המציג את התפלגות העובדים לפי מחלקות.
- **מדד מספרי** המציג את הגיל הממוצע של העובדים.

---

## 🚀 טכנולוגיות ושפות

- **Python 3.9+**
- **Streamlit**
- **Pandas**
- **NumPy**

---

## 📦 התקנה והרצה לוקאלית

1. **שכפול הריפו (Clone)**:

```bash
git clone <REPO_URL>
cd project10
```

2. **יצירת סביבת עבודה וירטואלית (אופציונלי אך מומלץ)**:

```bash
python -m venv .venv
.venv\Scripts\activate  # ב-Windows
```

3. **התקנת התלויות**:

```bash
pip install streamlit pandas numpy
```

4. **הרצת הדאשבורד**:

```bash
streamlit run dashboard.py
```

לאחר ההרצה, Streamlit יפתח אוטומטית את הדפדפן או יציג לך קישור מקומי בסגנון:

```text
http://localhost:8501
```

## 🧩 מבנה הפרויקט

- `dashboard.py` – קובץ הדאשבורד הראשי: יצירת נתונים מדומים, טבלה, גרף עמודות ומדד הגיל הממוצע.
- `dashboardRoey.py` – קובץ דוגמה בסיסי (כרגע רק `print("Hello World")`).
- `viz.py` – קובץ ריק שמיועד להרחבות ויזואליזציה נוספות בעתיד.

---

## 📚 איך הדאשבורד עובד?

1. **יצירת נתונים** – נבנה `DataFrame` של `pandas` עם נתוני עובדים (שם, גיל, מחלקה).
2. **הצגת טבלה** – הצגת הנתונים ב-TABLE אינטראקטיבי באמצעות `st.dataframe`.
3. **פילוח מחלקות** – ספירת העובדים בכל מחלקה והצגת התוצאה ב-bar chart באמצעות `st.bar_chart`.
4. **חישוב ממוצע גיל** – חישוב ממוצע הגילים והצגתו כמדד (`st.metric`).

ניתן בקלות להרחיב את הקוד ולהוסיף:
- פילוחים נוספים (למשל לפי ציון),
- פילטרים אינטראקטיביים,
- גרפים נוספים,
- טעינת נתונים מקובץ CSV אמיתי במקום נתונים מדומים.
